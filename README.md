# 🛒 Retail Sales Analytics Dashboard

## 📖 Project Overview

This project focuses on analyzing retail sales data using SQL and Power BI to generate actionable business insights. The dataset was cleaned and prepared for analysis, imported into a SQL database for business metric extraction, and visualized through an interactive Power BI dashboard.

The objective was to evaluate sales performance, profitability, product trends, customer segments, and regional performance to support data-driven business decisions.

---

## 🔄 Project Workflow

### Week 1: Data Cleaning & Preparation

* Cleaned the Superstore dataset using Python.
* Removed duplicate records.
* Handled missing values.
* Standardized data formats and column names.
* Validated sales, quantity, and profit fields.
* Exported the cleaned dataset for analysis.

### Week 2: SQL Business Analysis

Performed business analysis using SQL queries.

#### Analysis Performed

* Total Revenue Calculation
* Total Profit Calculation
* Category Performance Analysis
* Sub-Category Performance Analysis
* Product Performance Analysis
* Best-Selling Products Identification
* Regional Sales Analysis
* State-wise Sales Distribution
* Customer Segment Analysis
* Shipping Mode Analysis
* Discount Impact Analysis
* Monthly Sales Trend Analysis

### Week 3: Power BI Dashboard Development

Developed an interactive dashboard to visualize business performance.

#### Dashboard Components

* KPI Cards (Sales, Profit, Orders)
* Sales Trend Analysis
* Category & Sub-Category Performance
* Product Performance Dashboard
* Regional Sales Analysis
* Customer Segment Analysis
* Discount Analysis
* Interactive Filters and Slicers

### Week 4: Business Insights & Recommendations

Analyzed dashboard findings and documented key business insights and recommendations based on sales performance, profitability, product trends, customer behavior, and regional performance.
